Do you love beading?

Beading is a technique of creating artistic handicrafts that was already known in ancient Egypt. We can include all the methods in which we use small beads, a thin needle and thread. Although some argue whether braids created with the use of the kumihimo disk and crochet-bead cords should be included in the beading, if we consider two elements necessary to pass the beading method, i.e. thread and small beads, we can safely consider them as matching this category. Undoubtedly, beading can be called bead embroidery and various types of stitches, such as peyote, brick stich or right angle weave, i.e. RAW.
